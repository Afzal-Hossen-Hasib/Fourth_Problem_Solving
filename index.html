Ans : The Difference Between localStorage And SessionStorage :
localStorage :
Localstorage Shared Between All Tabs And Window From The Same Orgin. The Local Storage Helps Store Data With No Expiration Date. The Data Will Not Be Deleted Event The Browser Is Closed. It Is Possible To Obtain Data Any Time.
SessionStorage :
Sessionstorage Is Similar To Localstorage. It Stores Data Only For One Time. The Data Is Deletd When The User Closes The Browser Tab. Once The User Closes The Window The Data Is Lost.
